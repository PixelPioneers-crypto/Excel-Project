Project Overview
This project focuses on analyzing job application data from multiple companies using Microsoft Excel. The goal is to convert raw data into a clear, interactive dashboard that enables users to track application status and understand hiring trends straightforwardly.

Objectives
- Analyze job application data efficiently
- Track application status across different companies
- Present insights in a visually understandable format

Tool 
- Microsoft Excel

Technologies Used
- Pivot Tables  
- Pivot Charts  
- Excel Functions  
- Filters and Slicers  

How the Project Works
1. Raw job application data is collected and cleaned.
2. Data is organized using Excel tables.
3. Pivot tables are created to summarize key metrics.
4. Charts and slicers are added to build an interactive dashboard.
5. Final insights are presented in a single dashboard view.

Insights Gained
- Identification of companies with the highest number of applications
- Understanding of application outcomes
- Comparison of job application trends across companies

Project Use Cases
- Practice of Data Analytics using Microsoft Excel
- Portfolio project for data analyst roles
